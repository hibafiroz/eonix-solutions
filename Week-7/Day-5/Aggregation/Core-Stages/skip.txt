7. $skip:

- $skip is used to ignore a number of documents. 
- Itâ€™s mostly used for pagination. 
- After sorting or filtering, we can skip some documents and then return the next set of records.

Example:
Skip first student
db.students.aggregate([
 { $sort:{ marks:-1 } },
 { $skip:1 }
])


> $skip + $limit:

 Page 2 (2 records per page)
db.students.aggregate([
 { $sort:{ marks:-1 } },
 { $skip:2 },
 { $limit:2 }
])


Meaning:
Sort students
Skip first 2
Show next 2
