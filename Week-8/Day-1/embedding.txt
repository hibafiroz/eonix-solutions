> Embedding JavaScript in JSX:

In JSX we can embed JavaScript by using curly braces. 
Inside the braces we can write JavaScript expressions like variables, calculations, function calls, or conditional operators. 
React will evaluate the expression and display the result in the UI.

Anything written inside { } is treated as JavaScript.

Example 1: Display a Variable
function App() {
  const name = "Hiba"

  return <h1>Hello {name}</h1>
}

Output: Hello Hiba

Here {name} is JavaScript inside JSX.

Example 2: Using Expressions
function App() {
  const a = 5
  const b = 3

  return <h1>{a + b}</h1>
}
Output: 8
React evaluates the JavaScript expression inside { }.

Example 3: Calling Functions
function App() {
  function greet() {
    return "Good Morning"
  }

  return <h1>{greet()}</h1>
}

Output:
Good Morning

Functions can also be used inside { }.

Example 4: Conditional Rendering
function App() {
  const isLoggedIn = true

  return <h1> {isLoggedIn ? "Welcome" : "Please Login"} </h1>
}

Output: Welcome

Here we used a ternary operator inside JSX.
