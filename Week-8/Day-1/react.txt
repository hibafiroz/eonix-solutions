> What is React?

React is a JavaScript library that was developed by the Facebook Software Engineer Jordan Walke.
It is used to build user interfaces, especially for single page applications.
It follows component-based architecture and also uses a virtual DOM for better performance.


> How does React Work?

React creates a Virtual DOM in memory. When there is a change in state or props, 
it creates a new Virtual DOM and compares it with the previous one. After finding the differences, 
React updates only the changed elements in the real DOM instead of updating the entire DOM.


> What are the advantages of React?

React provides better performance using virtual dom, 
supports reusable components through component-based architecture, 
It helps developers to build interactive single-page applications.


> What are the limitations of React?

React focuses only on the UI layer, so additional libraries are needed for full application development. 
Also, the ecosystem changes frequently, and beginners may initially find JSX difficult.


> Why React?

- React uses a Virtual DOM which updates only the necessary parts of the UI instead of reloading the entire page, 
so the performance is better compared to traditional DOM manipulation.
- React follows a component-based architecture, so we can divide the UI into reusable components. 
This helps in scaling large applications. 


> Library vs Framework:

- A library is a collection of functions that we use when needed, and we control the flow of the program. 
- A framework provides a structure for building applications, and it controls the flow of the program.


> What is Create React App?

Create React App is a tool used to set up a React project quickly. 
It automatically configures things like Webpack and Babel so we can start 
building React applications without setting up the environment manually.


Note:

- The latest version of React is React 19 (around 19.2.x).
- React was first released in 2013 as an open-source JavaScript library.
- React was first used internally in Facebook’s News Feed in 2011, and later used in Instagram in 2012.
- React native used for mobile development
  Reactjs used for Web development
- React represents the view layer in the Model–View–Controller (MVC) architecture.  It is responsible for building and displaying the user interface.
