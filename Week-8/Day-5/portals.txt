> Portals:

React Portal is used when we want to render a component outside the parent DOM structure.
For example, if we render a modal inside a parent component, it may affect parent styles like overflow hidden, z-index, or positioning.
So we use React Portal to render the modal outside the parent DOM hierarchy using ReactDOM.createPortal.
Even though it is rendered outside the DOM, it still behaves like a normal React child and event bubbling still works.


> How Portals Work:

- Step 1 — Add a div in index.html
<div id="root"></div>
<div id="portal-root"></div>

- Step 2 — Create Portal
import React from "react";
import ReactDOM from "react-dom";


function Modal() {
  return ReactDOM.createPortal(
    <div className="modal">
      <h1>This is Modal</h1>
    </div>,
    document.getElementById("portal-root")
  );
}
export default Modal;

Now Modal renders in portal-root, not inside App.


> Why not render modal normally? Why Portal?
If we render modal inside parent, it may affect CSS issues like overflow hidden or z-index. 
So we use Portal to render it outside the parent DOM.
